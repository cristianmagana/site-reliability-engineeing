# Site Reliability Engineering

A comprehensive collection of SRE documentation, best practices, and reference materials.

## Documentation

This project uses MkDocs to generate documentation. To build and serve the documentation locally:

```bash
mkdocs serve
```

To build the static site:

```bash
mkdocs build
```

## Contents

- **Terminology** - Common SRE terminology and concepts
- **Databases** - Database concepts and best practices  
- **Linux** - Linux commands, troubleshooting, and system administration
- **Behavioral** - Interview preparation using the STAR method
- **Networking** - Network concepts and troubleshooting
- **Security** - Security best practices and considerations
- **Monitoring** - Monitoring strategies and tools
- **Automation** - Automation tools and practices
- **CI/CD** - Continuous Integration and Deployment practices
